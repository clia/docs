获取工具


npm install -g polymer-cli
mkdir my-app
cd my-app
polymer init

Polymer CLI 是一个命令行界面，用来使用 Polymer 构建元素，用 Polymer App 工具箱构建应用。它简化了工作流程，使从简单的组件到复杂的 Web App 项目易于管理。
